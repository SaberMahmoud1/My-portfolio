### **Prerequisites**

### **Installation Steps**

```bash
# 1️⃣ Clone the repository
git clone https://github.com/SaberMahmoud/My_Portfolio.git

# 2️⃣ Navigate to project directory
cd My_Portfolio

# 3️⃣ Install dependencies
npm install

# 4️⃣ Start development server
npm run dev

# 5️⃣ Open in browser
# Navigate to http://localhost:3000
```
